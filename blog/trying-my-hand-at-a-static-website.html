Trying my Hand at a Static Website

14 April 2024

I've been using WordPress to run my blog since I started it in 2011. Initially I used WordPress dot com, but in 2018 I switched over to hosting a WordPress dot org site. Lately though I've gotten tired of WordPress' cruft and have been longing for a much simpler static site.

Inspired by Louie Mantia Jr's How to Make a Damn Website post, here is my first post to the new site. I'm currently using GitHub Pages to host it because it's free and I'm not ready to attempt to port over 13 years of posts from my WordPress blog just yet.

Who knows? Maybe I'll even learn some HTML and CSS along the way. Fingers crossed.
